## Discord's Home Page Clone

### Features:
 - It's the user interface clone of the Discord's Home page.
 - The website UI is mobile & laptop screen friendly.
 - Mobile version of the UI has the hamburger menu.
 
### Tech Stack Used:
1. HTML 
2. Tailwind CSS
3. Javascript
